  • customPayload

    The custom payload associated with the bug. You may set any keys and values that you see fit. The values must be convertible into a JSON object using JSONSerialization.jsonObject(with:, options:).

    The following keys are used internally and will be silently ignored:

    • device_name
    • os_version
    • os_type
    • app_version
    • app_build_number
    • connection
    • available_storage
    • total_storage
    • current_screen

    If you wish to include keys from the list above, it is recommended to create a nested dictionary.
