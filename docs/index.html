Garage-ios

Project

Garage is a Swift Framework that helps product managers and the QA team to track bugs and send them to Jira.

Shake your device and Garage will appear. On iPad double tap with three finger on the screen.

Requirements

  • Swift 3+
  • iOS 9.3+
  • XCode 8+

Installation

Garage can be installed in two ways, either using the public version of the framework or directly pointing at this repository (for debugging purposes):

  • To use the public version of the framework, please follow the instructions in the public readme.
  • To use the private version of the framework (with source code), add the following line to your Podfile:

The recommended way to add Garage to your project is with Cocoapods. Add pod 'Garage', :git => 'ssh://git@github.com/Fueled/garage-ios.git' line to your Podfile and proceed with running pod install afterwards.

Please ensure that ssh is being used in this case, as it’s a requirement for our continuous integration setup.

After garage is installed (using either way), make sure that keychain sharing is enabled and that the group com.fueled is added to the list:

  • Your project needs to have the Keychain Sharing Enabled and you need to add com.fueled to the list:

Additionally, for Keychain Sharing to work as expected, the project has to have as bundle ID: com.fueled.<XXX> and be signed with a provisioning profile from the Fueled Enterprise account.

Usage

Please refer to the instructions in the public readme for information.

Development

GarageExample is the project used to develop Garage. It is linked to the CRIC Jira board.

Releasing

To release a new version of Garage, make sure you have been added as a Developer to the bitrise project, and have a build trigger token for the project. You may then run the following command:

curl https://app.bitrise.io/app/84b70b1d5e73bacb/build/start.json --data '{"hook_info":{"type":"bitrise","build_trigger_token":"<build trigger token>"},"build_params":{"branch":"develop","workflow_id":"deploy-public","environments":[{"mapped_to":"PUBLIC_VERSION_NUMBER","value":"<version number>","is_expand":true}]},"triggered_by":"curl"}'
