Garage-ios

Project

Garage is a Swift Framework that helps product managers and the QA team to track bugs and send them to Jira.

Shake your device and Garage will appear. On iPad double tap with three finger on the screen.

Requirements

  • Swift 3+
  • iOS 9.3+
  • XCode 8+
  • Your project needs to have the Keychain Sharing Enabled and you need to add com.fueled and com.Fueled to the list:

  • Your project has to have as bundl Id : com.Fueled.XXX and be signed with a provisioning profile from the team.

The last two points comes from the fact that the keychain is shared across all of our apps. If this requirement in not fulfilled the user will have to login again (And you don’t want to mess with your Product Manager!!).

Installation

You also need to ask your PM (or Paul) to provide a client id for the Snapshot and the Release configuration of your project. You will have to provide the bundle ID, JIRA name and configuration of your project.

Cocoapods

The recommended way to add Garage to your project is with Cocoapods. Add pod 'Garage', :git => 'ssh://git@github.com/Fueled/garage-ios.git' line to your Podfile and proceed with running pod install afterwards.

Please ensure that ssh is being used, as it’s a requirement for our continuous integration setup.

Carthage

The recommended way to add Garage to your project is with Carthage. Add github "Fueled/Garage-ios" line to your Cartfile and proceed with your favorite way to update and integrate your Carthage dependencies.

If submodules are being used for Carthage, please ensure that ssh is being used, as it’s a requirement for our continuous integration setup.

Example Project

You will find examples of using Garage in the GarageExample project.

Usage

Just call Garage.shared.configure in the AppDelegate in:

func application(_ application: UIApplication, didFinishLaunchingWithOptions launchOptions: [UIApplicationLaunchOptionsKey : Any]? = nil) -> Bool

You may call it like so:

Garage.shared.configure(clientId: <ClientID>, window: self.window)

Where ClientID is the client id for the configuration for the current project.

Releasing

To release a new version of Garage, make sure you have been added as a Developer to the bitrise project, and have a build trigger token for the project. You may then run the following command:

curl https://app.bitrise.io/app/84b70b1d5e73bacb/build/start.json --data '{"hook_info":{"type":"bitrise","build_trigger_token":"<build trigger token>"},"build_params":{"branch":"develop","environments":[{"mapped_to":"PUBLIC_VERSION_NUMBER","value":"<version number>","is_expand":true}]},"triggered_by":"curl"}'
